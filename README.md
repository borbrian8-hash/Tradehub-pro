# Tradehub-pro# TickFrame Pro

A professional trading platform for Deriv users featuring automated bots, live trading, and advanced analysis tools.

## 🚀 Features

- Secure Deriv OAuth Login
- Live Trading Dashboard
- Trading Bots
- Strategy Builder
- Tick Analysis
- Live Charts
- Portfolio Management
- Risk Management
- Trade History
- Performance Analytics
- Notifications
- Dark Mode
- Mobile Responsive

## 🛠 Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express
- TypeScript

### Database
- PostgreSQL
- Prisma ORM

### API
- Official Deriv WebSocket API
